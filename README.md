Edge AI terminal for IIoT with multi-modal SNN inference on 3x ESP32-S3. &lt;5W, 96%+ fault diagnosis accuracy, &lt;100ms safety interrupt. Supports condition monitoring, vision-based safety, LVGL HMI, MQTT cloud, LLM integration. 基于三片ESP32-S3的多模态SNN边缘AI终端，功耗&lt;5W，故障诊断准确率96%+，支持设备监测、视觉安全红区、LVGL交互、MQTT上云与大模型运维。
